## Commands to run this project

This project runs default in port 5173

http://localhost:5173

By the way the commands below in the same order meens:

- Install dependencies
- Run project

With pnpm:

```
pnpm i
pnpm dev
```

With npm:

```
npm i
npm run dev
```

With yarn:

```
yarn install
yarn run dev
```
